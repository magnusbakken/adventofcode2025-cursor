# Day 1 Part 1 - Advent of Code 2025

## Prompt
Try to solve day 1 part 1. The task file and the input file are present in the root of the repository.

## Solution
Created `day01.fsx` to solve the safe dial puzzle:
- Start at position 50 on a dial with numbers 0-99
- Follow rotation instructions (L for left/lower, R for right/higher)
- Count how many times the dial points at 0 after any rotation
- Handle circular wrapping (dial goes from 99 to 0 and vice versa)
